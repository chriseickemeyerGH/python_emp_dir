Employee Directory Python Application

This project is meant to be an interface for creating and/or updating a file containing employee directory data.

To run, run file 'src/Main.py' and follow the directions given by the app in the command line.

Other classes used in 'src/Main.py' are also inside 'src' as their own file
and any file read or written to while running the application will sit inside this directory as well.

To test, visit the 'tests' directory and run respective tests (35 total).
